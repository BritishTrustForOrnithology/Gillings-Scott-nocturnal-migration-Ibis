These folders contain code relating to Gillings & Scott 2021 Nocturnal flight calling behaviour of thrushes in relation to artificial light at night. Ibis

The Python folder contains code used to train neural networks.

The R code contains R code used to model the relationship between thrush call rates and artificial light. The associated dataset can be found at https://doi.org/10.6084/m9.figshare.14182799.v1
